Specs

A service for exposing specification files in version control via a nice web UI

View on GitHub Download .zip Download .tar.gz

Make your specifications visible

Specs is a service that allows you to make specifications written in Gherkin (Cucumber) and stored in Git easily visible to people who don't want to dive into Git. Specs is in very early development, if you have a feature request please raise it in GitHub, or even better create a pull request.
